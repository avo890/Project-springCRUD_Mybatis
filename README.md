##3차 프로젝트 개발기간 및 개발환경

#개발기간
-2023.06.30~2023.07.12

#개발환경
-spring Framework -4.3.18 release
-STS-3.9.11
-Oracle sql Developer-20.4.1
-apache-maven-3.9.2
-jsp
-Window10

#사용언어
-JS
-java jdk 1.8.0_361
-Oracle Database 11g Express Edition
-jQuery

#서버
-apache Tomcat -9.0.74

##주요기능
-로그인(로그아웃)
-아이디 및 비밀번호찾기
-상품 게시판 관리
-회원 관리
-댓글 관리(일정 상 비동기식이 아닌 동기식)
